  • Word Swipe

    link to git

    A gaming application where the user must identify english words and accumulate as many points as possible. This game is suitable for children between the ages of 5-12, every user can improve his or her english skills, in an enjoyable way while learning new words.
    This app is scalable to the users english level so any words that were left unanswered will repeat in the next round. Practice and sharp your english skills search and find new words in the gameboard.
    WordSwipe uses Firebase as its backend in order to keep track on every user's progress.
